This is a full-stack application — start the server to use it

The PMMS app is now a Node.js + SQLite web application. Opening this static HTML file directly does not load the app — you need to launch the server first.

1
Install Node.js (one-time) from nodejs.org — version 18 or newer.
2
Open a terminal in this folder and install dependencies:
npm install
3
Start the server:
npm start
Or just double-click start.bat on Windows.
4
Open the app: http://localhost:3000
Default login: admin / admin123
Full instructions and API reference are in README.md.
